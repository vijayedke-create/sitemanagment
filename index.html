☁️ Firebase Configuration
🔧 Setup (2 minutes, completely free):
  1. Go to console.firebase.google.com → Create project
  2. Build → Realtime Database → Create Database
  3. Choose "Start in test mode" → Enable
  4. Copy the Database URL shown (looks like:
    https://your-app-default-rtdb.firebaseio.com)
  5. Paste it below and click Connect
⚠️ Paste ONLY the URL (starting with https://) — no label text, no quotes
